Transitions: definition

Transitions define how an element change between two states. Different states may be defined using pseudo-classes like :hover or :active or dynamically set using JavaScript.

The transition CSS property is a shorthand property for transition-property, transition-duration, transition-timing-function, transition-delay, and transition-behavior[*].
height: 125px;
color: #202020;
background: #f09030;
border-radius: 10px;
transition: all 0.5s ease-in-out 1s;
/* 'transition' is the shorthand for: */
transition-property: all;
transition-duration: 0.5s;
transition-timing-function: ease-in-out;
transition-delay: 1s;
/* hover */
height: 250px;
color: #f09030;
background: #202020;
border-radius: 50%;

NOTE:
The special value all, which specifies that the transition will be applied to all properties that change as the element changes state.
