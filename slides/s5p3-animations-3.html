Animations examples

@keyframes move {
 0% {
  left: 0%;
  transform: translate(-50%, -50%);
  background: red;
 } 33% {
  left: 45%;
  background: yellow;
 } 50% {
  left: 50%;
  background: green;
 } 66% {
  left: 55%;
  background: cyan;
 } 100% {
  left: 100%;
  transform: translate(-50%, 50%);
  background: blue;
 }
}

NOTE: In examples #5 and #6, the transform functions of the two animations do not combine; the second one overrides the first.

animation: move 3s ease-in-out infinite paused; //# 1
animation: move 3s ease-in-out infinite reverse paused; //# 2
animation: move 3s ease-in-out infinite alternate paused; //# 3
animation: move 3s ease-in-out infinite alternate-reverse paused; //# 4
animation: move 3s ease-in-out infinite alternate paused, //# 5
  pulse 0.5s 1s ease-in-out 10 paused;
animation: pulse 0.5s 1s ease-in-out 10 paused, //# 6
  move 3s ease-in-out infinite alternate paused;
/* hover */
animation-play-state: running;
