Transform origin

The transform-origin CSS property sets the origin for an element's transformations: the point around which a transformation is applied. This property may be specified using one, two, or three values, where each value represents an offset. By default, the origin is set to center.
transform-origin: top right; // #1
transform-origin: center; // #2
transform-origin: 25% 75%; // #3
transform-origin: center center 250px; // #4
/* transformations on hover */
transform: rotate(45deg); // #1, #2, #3
transform: rotateY(45deg); // #4

NOTE: the red dot represents the transform origin point.
