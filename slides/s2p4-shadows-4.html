Multiple box shadows

box-shadow accepts multiple effects, separated by commas, to obtain layered shadows.
Multiple shadows
/* long shadow */
box-shadow:
 0px  3px  3px       rgba(0, 0, 0, 0.05),
 0px  6px  6px       rgba(0, 0, 0, 0.07),
 0px 12px 12px  -6px rgba(0, 0, 0, 0.1),
 0px 24px 24px -12px rgba(0, 0, 0, 0.3),
 0px 48px 48px -24px rgba(0, 0, 0, 0.5);
/* "creative" effects */
box-shadow:
   0px -25px 20px hsl(  0 100% 50% / 0.5),
  13px -13px 20px hsl( 45 100% 50% / 0.5),
  25px   0px 20px hsl( 90 100% 50% / 0.5),
  13px  13px 20px hsl(135 100% 50% / 0.5),
   0px  25px 20px hsl(180 100% 50% / 0.5),
 -13px  13px 20px hsl(225 100% 50% / 0.5),
 -25px   0px 20px hsl(270 100% 50% / 0.5),
 -13px -13px 20px hsl(315 100% 50% / 0.5);
