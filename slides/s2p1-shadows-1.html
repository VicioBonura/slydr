Shadows: definition

The box-shadow CSS property adds shadow effect to an element's frame.
The text-shadow CSS property adds shadow effect to text.
/* box-shadow */
div {
 box-shadow: 2px 3px 5px #888888;
}
/* text-shadow */
p {
 text-shadow: 2px 3px 5px #888888;
}
