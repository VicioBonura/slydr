Gradients: definition

Gradient CSS Function creates an image consisting of a progressive transition between two or more colors. There are 3 types of gradients: linear-gradient, radial-gradient and conic-gradient.
/* linear-gradient */
background-image:
 linear-gradient(to right, #ffffff, #3090f0);
/* radial-gradient */
background-image:
 radial-gradient(circle at center, #ffffff, #3090f0);
/* conic-gradient */
background-image:
 conic-gradient(from 0.5turn at center, #ffffff, #3090f0);
