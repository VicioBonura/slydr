Transformations: definition

The transform CSS property is used to rotate, scale, skew, or translate an element, in a two-dimensional plane or in 3D space. This property may be specified as either the keyword value none or as one or more transform-function values.
/* 1: translate */
transform: translate(-30px, 10px);
/* 2: rotate */
transform: rotate(15deg);
/* 3: scale */
transform: scale(1.25);
/* 4: skew */
transform: skew(15deg, 15deg);
