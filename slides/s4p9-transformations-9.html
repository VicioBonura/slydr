Functions VS properties

/* transform functions */
transformn: translate(50px, 10px) scale(1.1);
/* rotate on hover */
transformn: translate(50px, 10px) scale(1.1) rotate(30deg);

Transformation functions are applied to the transform property in a single declaration. To change one transformation, you must also specify, in the correct order, the functions that are not modified.
Remember: order is important. The effects are applied from left to right.

/* transform properties */
translate: 50px 10px;
scale: 1.1;
/* rotate on hover */
rotate: 30deg;

Transform properties allow for more granular control over each transformation. However, the order still matters and is always the same, Regardless of how they are declared: first translate, then rotate, and then scale. Since these properties were introduced recently (2022), compatibility with older browsers may not be guaranteed.
