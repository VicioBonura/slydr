Timing

Duration

This property sets the length of time a transition animation should take to complete. By default, the value is 0s, meaning that no animation will occur.

width: 0%;
transition: width 5s;
/* hover */
width 100%;
Duration

Delay

This property specifies the duration to wait before starting a property's transition effect when its value changes. By default, the value is 0s, meaning that the transition will start immediately.

transition: width 5s 2s;
Delay
transition: width 5s;
/* hover */
transition-duration: 1s;
Duration
transition: width 5s 2s;
/* hover */
transition-delay: 0s;
