Let's recap

transform: <function comma-separated list>;
/* some functions */
transform: translate( <offset-x>, offset-y );
transform: rotate( <angle-z> );
transform: scale( <number> );
transform: skew( <angle-x>, <angle-y> );
transform: perspective( <length> );
  • Transformation functions can modify an element in a three-dimensional space
  • The order in which the functions are declared is important
  • transform-origin sets the origin for an element's transformations
  • New transform independent properties are more powerful, but less compatible with older browsers
