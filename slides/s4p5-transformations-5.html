Rotate

  • The rotate() CSS function defines a transformation that rotates an element around a fixed point on the 2D plane (z=0 plane).
  • The rotate3d() CSS function defines a transformation that rotates an element around a fixed axis in 3D space. The axis of rotation is defined by an [x, y, z] vector and pass by the origin (as defined by the transform-origin property).
/* parent */
transform: perspective(500px);
perspective-origin: center;
transform-style: preserve-3d;
/* child: hover */
transform: rotate(45deg); // #1 - equal to rotateZ(45deg)
transform: rotateX(-1.85turn); // #2
transform: rotateY(135deg); // #3
transform: rotate3d(1, 1, 1, 90deg); // #4
transform: rotateX(45deg) rotateY(65deg) rotateZ(30deg); // #5
transform: rotateY(135deg); // #6
backface-visibility: hidden; // #6
