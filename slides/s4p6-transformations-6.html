Scale and Skew

  • The scale() CSS function defines a transformation that resizes an element on the 2D plane. The function takes two arguments: the horizontal and vertical scale factors; if only 1 argument is provided, the scale is applied to both axes.
  • The scale3d() CSS function defines a transformation that resizes an element in 3D space.
  • The skew() CSS function defines a transformation that skews an element on the 2D plane. The function takes two arguments: the horizontal and vertical skew factors; if only 1 argument is provided, the skew is applied only to the X-axis.
/* scale on hover */
transform: scale(1.2); // #1
transform: scale(0.8, 1.2); // #2
transform: scale(-1, 1); // #3
/* skew on hover */
transform: skew(20deg); // #4
transform: skew(20deg, 10deg); // #5
transform: skew(40deg, -40deg); // #6
