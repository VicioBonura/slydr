Easing

The transition-timing-function CSS property sets how the intermediate values of CSS properties are calculated during a transition. The function represents an acceleration curve and is defined using one easing-function for each property to be transitioned.
/* linear functions */
transition-timing-function: linear;
transition-timing-function: linear(0,0.25 75%,1);
/* cubic b√©zier function */
transition-timing-function: ease-in;
transition-timing-function: ease-out;
transition-timing-function: ease-in-out;
transition-timing-function: cubic-bezier(0, 1.6, 1,-0.6);
/* steps function */
transition-timing-function: step-start;
transition-timing-function: steps(5, end);

Hover over the dashed area to start the transitions
