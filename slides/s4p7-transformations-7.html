Multiple transformations

The transform property takes one or more tranformations function. If multiple transformations are applied to the same element, the effects are applying in the order from left to right.
/* translate and rotate */
transform: translate(30px) rotate(45deg); // #1
transform: rotate(45deg) translate(30px); // #2
/* translate and rotate */
transform: scale(0.5, 1.5) rotate(90deg); // #3
transform: rotate(90deg) scale(0.5, 1.5); // #4
/* 3D tranformations */
transform: translateZ(100px) rotateY(45deg); // #5
transform: rotateY(45deg) translateZ(100px); // #6
