Anatomy of a statement

background-image: linear-gradient(30deg, orange 0%, cyan 50%, lime 100%);
background-image: conic-gradient(from 30deg at center, orange 0%, cyan 50%, lime 100%);
background-image: radial-gradient(circle farthest-side at center, orange 0%, cyan 50%, lime 100%);

Values

  • function: generates the image according to the parameters passed as arguments
  • direction:
    • Linear: the direction of the gradient line
    • Conic: the starting point of the clockwise rotation of the gradient
  • shape: the shape of the gradient (circle or ellipse)
  • size: size of the gradient's ending shape
  • position: the position of the gradient's starting point
  • color-stop list: a list of color-length couples
    • color: the color value at the selected point of the gradient
    • length: the length or percentage of the color along the gradient line
