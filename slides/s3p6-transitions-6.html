Let's recap

/* shorthand */
transition: <property> <duration> <timing-function> <delay>;
/* separated properties */
transition-property: <property-ident>;
transition-duration: <time>;
transition-timing-function: <easing-function>;
transition-delay: <time>;
  • A transition occurs when the state of an element changes, which can be triggered by pseudo-classes (e.g., :hover or :focus) or dynamically by JavaScript.
  • All CSS properties are animatable unless otherwise specified.
  • We can chain multiple transitions by separating them with a comma.
