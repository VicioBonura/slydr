Translate

  • The translate() CSS function repositions an element in the horizontal and/or vertical directions.
  • The translate3d() CSS function repositions an element in 3D space. Its result is a data type.
/* parent */
transform: perspective(500px);
perspective-origin: center;
transform-style: preserve-3d;
/* child: hover */
transform: translate(20px); // #1
transform: translate(20px, 10px); // #2
transform: translateX(-20px) translateY(10px); // #3
transform: translateZ(-150px); // #4
transform: translate3d(0px, 0px, 150px); // #5
transform: translate3d(-10px, -10px, 200px); // #6
