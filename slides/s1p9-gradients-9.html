Let's recap

background-image: linear-gradient( <direction|angle>, <stop-color-list> );
background-image: radial-gradient( <shape> <size> <position>, <stop-color-list> );
background-image: conic-gradient( <angle> <position>, <stop-color-list> );
  • We can create a pattern and repeat it with repeating-gradients
  • We can use more than one gradient by separating them with a comma
  • We can choose the interpolation method between colors
  • Gradients are images: we can use background properties for advanced effects
